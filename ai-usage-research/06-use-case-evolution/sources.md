# Sources: Use Case Evolution Over Time

## Primary Temporal Research

### OpenAI ChatGPT Usage Study
- **URL**: https://openai.com/index/how-people-are-using-chatgpt/
- **PDF**: https://cdn.openai.com/pdf/a253471f-8260-40c6-a2cc-aa93fe9f142e/economic-research-chatgpt-usage-paper.pdf
- **Temporal Data**:
  - Work vs personal: 53% non-work (mid-2024) → 72% non-work (mid-2025)
  - Technical help: 12% (July 2024) → 5% (July 2025)
  - Message volume: 451M/day (June 2024) → 2.6B/day (June 2025)

### Anthropic Economic Index
- **URL**: https://www.anthropic.com/research/anthropic-economic-index-september-2025-report
- **Temporal Data**:
  - Automation: 27% (late 2024) → 39% (August 2025)
  - Educational tasks: 9% (early 2025) → 12% (August 2025)
  - Scientific tasks: 6% (early 2025) → 7% (August 2025)
  - Coding: 36% consistent

### Microsoft Copilot Usage Report
- **URL**: https://microsoft.ai/news/its-about-time-the-copilot-usage-report-2025/
- **Temporal Data**:
  - January → September 2025 evolution
  - Fewer programming conversations
  - More culture and history activity
  - Seasonal patterns (Valentine's Day, commuting hours, etc.)
