# Sources: Automation vs Augmentation Patterns

## Primary Research Reports

### Anthropic Economic Index (September 2025)
- **URL**: https://www.anthropic.com/research/anthropic-economic-index-september-2025-report
- **Published**: September 15, 2025
- **Data**: Task-level usage patterns for Claude.ai and enterprise API
- **Key Contribution**: Automation vs augmentation definitions and temporal trends
- **Dataset**: Open-sourced

### OpenAI ChatGPT Usage Study (September 2025)
- **URL**: https://openai.com/index/how-people-are-using-chatgpt/
- **PDF**: https://cdn.openai.com/pdf/a253471f-8260-40c6-a2cc-aa93fe9f142e/economic-research-chatgpt-usage-paper.pdf
- **Published**: September 17, 2025
- **Researchers**: OpenAI + Harvard economist David Deming
- **Data**: 1.5 million conversations analyzed
- **Key Contribution**: Work vs personal usage patterns, message categorization

### Microsoft Copilot Usage Report (December 2025)
- **URL**: https://microsoft.ai/news/its-about-time-the-copilot-usage-report-2025/
- **Published**: December 10, 2025
- **Data**: 37.5 million conversations (January-September 2025)
- **Key Contribution**: Desktop vs mobile interaction patterns, contextual usage

### Microsoft New Employee Research (April 2025)
- **URL**: https://www.microsoft.com/en-us/research/publication/new-employee-copilot-usage-insights-into-productivity-and-socialization/
- **PDF**: https://www.microsoft.com/en-us/research/wp-content/uploads/2025/04/New-employee-Copilot-usage.pdf
- **Participants**: 125 Microsoft interns
- **Methods**: Surveys, interviews, diary studies
- **Key Contribution**: Productivity quantification, use case ranking

### Microsoft Work Trend Index
- **URL**: https://www.microsoft.com/en-us/worklab/work-trend-index/copilots-earliest-users-teach-us-about-generative-ai-at-work
- **Key Contribution**: Early adopter productivity metrics (70% more productive, 29% faster)
